🔐 Your Formspree Account

📌 How to create a second form: Log in to Formspree, click “+ New form”, name it “Consultation Requests”, set your email, and copy the endpoint URL.

⏱️ How Long Can You Use Formspree?

Forever, if you want. The free plan gives you 50 submissions per month across all forms – perfect for starting out. If you ever need more, you can upgrade or later switch to your own backend. The form code stays exactly the same; only the action URL would change.

📍 Where to Paste the Modal Code

You must include the consultation modal HTML on every page where the “Free Consultation” button appears (usually all pages). Put it just before the closing </body> tag, after your footer.

Also add the small CSS block (under “CSS to add”) to your global styles.

🎨 CSS to Add (inside your <style> tag)

Add these rules if you haven’t already. They keep the checkbox aligned and inputs from overflowing.

/* For consultation modal */
.modal-form input,
.modal-form textarea {
    box-sizing: border-box;  /* prevents width overflow */
}

.modal-checkbox {
    display: flex;
    align-items: center;
    gap: 10px;
    margin-bottom: 15px;
}
.modal-checkbox input[type="checkbox"] {
    width: auto;
    margin: 0;
}
.modal-checkbox label {
    color: white;
    margin: 0;
}
.modal-checkbox a {
    color: var(--saffron);
}

⚙️ JavaScript – What to Keep / Remove

In your existing script, remove this one line (it blocks Formspree):

document.getElementById('consultForm')?.addEventListener('submit', (e) => { e.preventDefault(); alert('Thank you!...'); hideModal(); });

Keep all the other modal-related JavaScript (open/close, next/back, time slots, flatpickr).
